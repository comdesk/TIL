<h1>Grid 개별 아이템 중앙 정렬</h1>
<hr>

<ul>
  <li>justify-self: 해당 아이템을 각 셀 안에서 수평 정렬 할 수 있다.</li>
  <li>align-self: 해당 아이템을 각 셀 안에서 수직 정렬 할 수 있다.</li>
</ul>

</ul>
